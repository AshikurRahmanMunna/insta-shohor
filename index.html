What Does Event Loop Do

Javascript event loop manages the code how to run. Javascript has 2 type of event. Synchronous Event and Asynchronous Event. Javascript is a single threaded application. So it can't run Synchronous & Asynchronous event at a time. Javascript code runs from the top to bottom. But when javascript is running some type of code runs asynchronously. Like fetch, setTimeout, setInterval etc. when the code is running from top to bottom the asynchronous events are added to a queue. So they runs after the entire code has completed execution. Javascript runs the first code of queue and then other codes. So we don't have to wait for the completion of asynchronous events.

Difference Between localStorage & SessionStorage

localStorage is a storage system that browser stores. localStorage is stored for accessing a data after long time. We can use localStorage by setting item to localStorage. And we can also get item from localStorage. localStorage is a permanent storage system. We can access this after browser window is closed or browser tab is closed. In the other hand sessionStorage is also a storage system that browser stores. sessionStorage is stored for accessing a data when the user is active on browser. We can use sessionStorage by setting item to sessionStorage. And we can also get item from sessionStorage. sessionStorage is not a permanent storage system. It is a runtime or temporary storage system. We can only access this when browser window is open or browser tab is opened.
